Imagine Board


DOCKER HEADER

Directory Open : Select Directory to explore image files inside.
Thumbnail Cache : Cache thumbnails of the selected directory within a dynamic range.
SlideShow Play/Stop : Play a slideshow of the selected directory.
Function Input : Input function or run the current function. Inactive widget displays the current operation and the required arguments be it strings(s.) and or numbers(n.). Type for manual input or click with RMB to see context menu of available commands.


DOCKER CONTROLS

Progress : Displays thumbnail cache progress relative to the Slider. Also files are cached by selection and range of proximity.
Slider : Select a given image within a directory using a slider.
Mode : Select the docker panel display. Displays are PREVIEW and GRID.
Filter Contents : Filters the files found within the directory by using keywords. Keywords can be positive or negative. Starting with positive keywords to create a selection and then use the word 'NOT' for negative keywords and negate from the selection.
Number Index : Select a given image within a directory using a number. Also displays quantity of all found files.
Settings : Open the settings dialog window.


SETTINGS

Filter Directory : Filter between NORMAL and krita BACKUP~ files.
Sort Directory : Sort files by several methods. Default setting is Local Aware, that considers the OS settings towards that folder.
SlideShow Sequence : Select between LINEAR and RANDOM sequences. LINEAR will go through all images as they were sorted by the filters. RANDOM will go through all images in random fashion and will only repeat until a cycle is completed.
SlideShow Interval : Select the amount of time spent on each image in seconds.
Grid : Select the amount of horizontal(U) and vertical(V) division blocks for the Grid display.


SETTINGS FUNCTION (KEYENTER)

File : Displays the selected file name and other information.
Operation : Select the Function Operation from a list.
String : Add strings to the list and then select the actives ones to be used as an argument. Selected strings can be deleted using the clear button.
Number : Set a start number for the counter to be used as an argument when using an operation with ORDER.
Path Text : Verbose text for a valid path (file or folder). On the SAVE ORDER operation this will be the input path and the DIRECTORY for display will be the output path so you can see the copies arriving.
Path Run : Only valid paths will enable the run button to apply the current function.


OPERATIONS

>> : Null effect.
Key Add : Add keywords to the filename. Requires a string for the keyword.
Key Replace : Replace all keywords from the filename. Requires a string for the keyword.
Key Remove : Remove keyword from the filename. Requires a string for the keyword.
Key Clean : Clean all keywords from the filename.
Key Echo : Echo search files to detect visually identical files. ECHO keyword replaces all keywords on the filename.
Rename Order : Rename files in numerical order. Requires a string for the filename and a number for the inicial value counter.
Rename Random : Rename files in random order. Requires a string for the filename and uses random alphanumeric sequences instead of numbers.
Save Order : Save files in numerical order to the panel display directory. Requires a string for the filename and a number for the inicial value counter. Drag and Drop files to save into the Display Directory. Use a Text Path and Display Directory to copy from one folder to another. Use Context to menu to duplicate files on the same folder.


FILE INPUTS

Function Input : Write 'run>>' to apply the function to the selected image.
Panel Context : Open the context menu and use the FUNCTION>> option to apply the function to the selected image.
Drag and Drop : Drag and Drop image files or folders from any source to apply the function to the dropped selection (images and or folders). Function applies to files only (even when populated with folders in the selection), or will apply to a selection with a single folder (with no files).
Path Text/Run : Input text for a valid file or folder path and hit the run button.


PREVIEW INTERACTION

LMB x1 : None.
LMB x2 : Change to Grid mode.
SHIFT + LMB : Incremental Browse.
CTRL + LMB : Mouse/Stylus Browse.
ALT + LMB : Drag and Drop file.

RMB x1 : Context Menu.
SHIFT + RMB : Pan image.
CTRL + RMB : Zoom image.
ALT + RMB : Reset Pan and Zoom.


GRID INTERACTION

LMB x1 : Display filename.
LMB x2 : Change to Preview Mode.
SHIFT + LMB : Incremental Browse.
CTRL + LMB : Mouse/Stylus Browse.
ALT + LMB : Drag and Drop file.

RMB x1 : Context Menu.
SHIFT + RMB : None.
CTRL + RMB : None.
ALT + RMB : None.


CONTEXT PREVIEW

Mode Grid : Change to Grid mode.
Function>> : Run Function to the current file.
Random : Select a Random file from the directory.
Location : Open the file location with the OS.
Copy : Copy text to the FILENAME, PATH DIRECTORY or PATH FULL.
Clip Image : Highlight a section of the image to clip it when using any insert command.
Fit Insert : Scale inserted image to document size.
New Document : Open file as a new Document.
Insert Reference : Insert file as a Reference.
Insert Layer : Insert file as a Layer.


CONTEXT GRID

Mode Preview : Change to Preview mode.
Function>> : Run Function to the current file.
Fit Ratio : Adjust thumbnail display.


EXTENSION SHORTCUTS

Browse Minus : Browse files into a lower number.
Browse Plus : Browse files into a higher number.
